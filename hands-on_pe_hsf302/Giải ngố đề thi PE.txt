* ĐỘ DÀI ĐỀ THI >= 5 TRANG VÀ 1 FILE DB SCRIPT 
  - ĐẦU ĐỀ THI CHỨA TÊN DỰ ÁN VÀ MÀN HÌNH CHÍNH CRUD TABLE X

  PHẦN ĐUÔI FILE DB SCRIPT CÓ DẠNG .TXT HOẶC .SQL
  (.TXT DOUBLE CLICK DEFAULT MỞ NOTEPAD
   .SQL DOUBLE CLICK DEFAULT MỞ SQL SERVER MANAGEMENT STUDIO NẾU CÓ CÀI)

    DB FIRST NGHĨA LÀ SẼ TỪ TABLE DỘI NGƯỢC RA @ENTTY CLASS

    TRÊN LỚP MÌNH ĐÃ HỌC "CODE FIRST" - TỪ @ENTITY CLASS SẼ THÀNH TABLE
