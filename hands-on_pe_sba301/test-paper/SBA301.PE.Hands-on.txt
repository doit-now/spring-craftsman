Đính kèm là đề thi mẫu môn học làm app React sử dụng Java Script kết hợp gọi API Spring Boot. Không sử dụng TypeScript  
Tôi nhờ bạn generate giúp tôi app React với các yêu cầu sau:
1. Về cấu hình app
- API không cần làm, hãy sử dụng các endpoint theo đầu bài
- Tuân thủ đầu bài về các thư viện và công nghệ sử dụng. Chỉ sử dụng các thư viện được đề xuất
- Thư viện sử dụng: React Boostrap, Axios, React Router DOM, Vite.
- Tất cả các thư viện dùng version mới nhất hiện nay. (Theo tôi được biết React đang là version 19.x hoặc 20.x)
- Code dự án dùng JavaScript, TUYỆT ĐỐI KHÔNG DÙNG TypeScript  
- Port API là 8080
- Port của React app là 5173
 
2. Về chức năng VÀ UI:
- Đây là app CRUD cặp table One to Many Category and Shoes, nhưng tính năng chính chỉ tập trung CRUD trên table Many Shoes
- Chức năng chi tiết bạn xem các hình mô tả trong đề thi
- Trong đề có thấy màn hình chính, Shoes List có phần phân trang, và API cũng đã có hỗ trợ phân trang pagination trên các tiêu chí: page, size, sort by field (default shoesName), sort by direction. Tôi đính kèm thông tin field trả về của phần phân trang ở mục 3 ở ngay dưới. Màn hình bạn chỉ cần làm kết quả điều hướng phân trang, không cần thiết kế phần lựa chọn page, size... trên màn hình.
- Giao diện tuân thủ đúng các field, layout mà đã chỉ ra trong hình đề thi và hình đính kèm tôi đã tách riêng
- Giao diện trang web DÙNG TIẾNG ANH như trong đề thi yêu cầu, đúng đủ field không tự thêm bớt.
- Về Action, bạn thêm link Edit, kế bên link View, màn hình Edit giống màn hình tạo mới Add New, nhưng khác màn hình View. Các màn hình tôi có đính kèm riêng theo prompt.
- Về validation làm cơ bản: tên sản phẩm không trùng, số trong 1 đoạn, các field required, ngày tháng năm theo dd/mm/yyyy, đừng làm code thêm phức tạp, giữ đủ đơn giản để cho người mới bắt đầu có thể tiếp cận code nhanh chóng. Validation hiển thị đỏ cạnh input (inline)
- Dùng bảng màu mặc định của React Boostrap, không chế thêm CSS để khiến code thêm phức tạp cho người mới bắt đầu
