I. ĐỀ THI CÓ GÌ LẠ?
1. >= 10 TRANG ĐỀ THI


2. 2 FILE SCRIPT CHỨA LỆNH SQL TẠO CÁC TABLE CHO 2 SERVICE (2 API PROJECT) KHÁC NHAU
   NHƯNG CÁC TABLE NÀY ĐANG TẠM Ở CHUNG 1 DATABASE NHƯNG ĐỘC LẬP KO RELATIONSHIP
   (1 DATABASE DÙNG CHUNG CHO 2 SERVICE CHO TIỆN LÀM BÀI)   
   * ĐÚNG CHUẨN NGOÀI CÔNG NGHIỆP MỖI SERVICE DÙNG 1 DATABASE RIÊNG, TABLE RIÊNG
   
   2 FILE SCRIPT CHỨA LỆNH TẠO 2 TABLE MỐI QUAN HỆ 1 - N (DEPARTMENT ----< EMPLOYEE)
   NHƯNG BÀI THI GIẢ LẬP CHÚNG KHÁC DATABASE NÊN KHÔNG CÓ KÉO RELATIONSHIP!!! 
  
   * LÁT HỒI TRONG ENTITY KHÔNG DÙNG ĐƯỢC @OneToMany, @ManyToOne TRONG CLASS DEPARTMENT, EMPLOYEE
                                                 DO 2 TABLE / ENTITY ĐANG GIẢ BỘ ĐỘC LẬP DATABASE
   >>>>> DATABASE FIRST: CÓ TABLE -> @ENTITY
