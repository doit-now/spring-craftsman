Ghi chú học nhanh:
- fetch(url) là cách browser gửi HTTP request.
- Status là mã phản hồi (200, 4xx, 5xx...).
- Body là nội dung server trả về (String hoặc JSON).
- Nếu bạn bị CORS, cấu hình Spring Boot cho phép origin http://localhost:3979 (hoặc port bạn serve).
